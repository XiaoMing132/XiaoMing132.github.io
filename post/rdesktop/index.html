linux连接Windows远程桌面

Nov 3, 2022
1 min read

##本文介绍的是在linux连接远程桌面，相对比较简单 ####连接远程桌面需要一个客户端，windows下面是mstsc。linux在这里我们使用rdesktop这个远程连接软件

1:下载软件，软件下载有很多种方法，我使用的是ubuntu系统，在这里我们直接使用apt进行安装

sudo apt install rdesktop

2:使用rdesktop命令进行远程连接

rdesktop -u username -p passwd IP

####注意，使用rdesktop命令需要你的Windows打开允许任意版本连接远程桌面，这一步大家就自行百度了
